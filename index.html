Macro Counter

Previous Next

Macro Counter is a hobby project I created to help educate myself and others to eat better foods daily. The application provides information on what macronutrients actually are, and also allows the user to input the calories, fats and carbs they eat, which are then converted into their actual macronutrient intake.

Built with: HTML5, CSS3, JavaScript, jQuery, and Bootstrap 4.

Deployed on Heroku.
